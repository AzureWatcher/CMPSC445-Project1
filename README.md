Navigating this repository:
*   All original datasets can be found in the `databases` folder. See the Project Report, section References (pg. 16) for information on the sources of these datasets from on the Internet.
*   `databases/full_database.csv` contains the full merged dataset after collecting and merging, but before preprocessing. As such, it contains numerous `NaN` values.
*   `databases/preprocessed_database.csv` contains the preprocessed version of the original merged dataset and is the version of the dataset used to train the various models implemented in `main.py`.
*   `CMPSC 445 Project 1 Report.pdf` is the written report for this project, and contains all plots and discussion required by the assignment brief.
*   `main.py` contains all the code for this project.
